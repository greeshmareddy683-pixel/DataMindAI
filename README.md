#  DataMind AI
Natural language database analyst powered by LLaMA 3.3-70b & Groq

## Features
-  Chat with your database in plain English
-  Auto-generates charts & diagrams
-  Voice input support
-  Custom dashboard builder
-  Query history & favorites
-  Multi-database support
